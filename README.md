# Chat-Front

## Descrição do Projeto
Este projeto foi desenvolvido como atividade para na matéria Desenvolvimento Rápido de Aplicações (DRA). A aplicação consiste em um sistema de chat, onde os usuários podem se comunicar em tempo real via mensagens.

## Tecnologias Utilizadas
### Backend:
- Spring Boot
- Java

### Banco:
- H2

### Frontend:
- Thymeleaf
- Java
- HTML
- CSS
